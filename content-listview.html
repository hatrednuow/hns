일반 요구사항 열에 노출된 탱크내의 화물은 비등액체-팽창 증기 폭발(BLEVE : Boiling Liquid - Expanding Vapour Explosion)에 의하여 화재 시 또는 화재 후 갑자기 폭발할 수 있음. 다량의 물을 사용하여 탱크를 냉각시킬 것. 가능한 멀리 떨어진 보호된 위치에서 화재를 진압할 것. 실행가능하다면 유출을 멈추게 하거나 개방된 밸브를 폐쇄시킬 것. 불꽃은 눈에 보이지 않을 수도 있음.
갑판 상부 화물화재 포장화물 : 가능한 한 많은 호스를 사용하여 물을 분사할 것. 화물운송장치 : 연소중인 화물운송 장치와 화재에 노출된 인접 화물을 다량의 물을 사용하여 냉각시킬 것.
갑판 하부 화물화재 환기를 중지하고 해치를 닫을 것. 화물구역에 설치된 고정식 소화장치를 사용할 것. 이것이 불가능하다면 다량의 물을 사용하여 물이 분사되도록 할 것.
화재 노출화물 실행 가능하다면 화재에 연루될 가능성이 있는 포장화물은 제거하거나 바다에 버릴 것. 그렇지 아니하다면 물을 사용하여 수 시간 냉각시킬 것.
특례 화물이 물과 접촉 시에는 염산(hydrochloric acid)을 생성할 것임 : 유출물에서 멀리 떨어질 것.

일반 요구사항 보호의 및 자장식 호흡구를 착용할 것. 보호의를 착용하였다고 하더라도 접촉을 피할 것. 유출되는 화물에 접근을 금할 것. 발생하는 증기에 접근을 금할 것. 심지어 아주 짧은 시간 소량의 증기를 흡입 시에도 호흡곤란을 야기할 수 있음. 해당 물질 표면에 물을 사용하면 격렬히 반응하여 독성 증기를 생성시킬 수 있음. 해당 물질은 선박의 구조물에 손상을 줄 수 있음. 유출물에서 또는 물과의 반응으로 인화성 증기를 방출시킬 수 있음. 모든 점화원을 피할 것(예, 나화, 보호되지 아니한 전구, 전동 수공구, 마찰 등). 오염된 의복은 물로 씻은 후 제거할 것.
갑판 상부 화물화재 포장화물 (소량유출) : 다량의 물을 사용하여 선외로 씻어 낼 것. 유출물 위에 직접 물을 강하게 분사하지 말 것. 유출되는 화물에 접근을 금할 것. 유출된 장소를 완전히 청소할 것. 화물운송장치 (대량유출) : 선교와 거주구역이 바람이 불어오는 쪽으로 향하도록 할 것. 물분사장치를 사용하여 부식성이나 독성 증기를 몰아냄으로써 승무원과 거주구역을 보호할 것. 다량의 물을 사용하여 선외로 씻어 낼 것. 유출물 위에 직접 물을 강하게 분사하지 말 것. 유출되는 화물에 접근을 금할 것. 유출된 장소를 완전히 청소할 것.
갑판 하부 화물화재 포장화물 (소량유출) : 적절한 환기조치를 취할 것. 자장식 호흡구를 착용하지 아니하고는 갑판으로 들어가지 말 것. 들어가기 전에 대기상태를 점검할 것(독성 및 폭발 위험성). 만약 대기상태를 점검할 수 없을 시에는 들어가지 말 것. 증기가 증발하도록 둘 것. 접근 금지. 액체 : 유출된 구역을 충분히 환기시킬 것. 선창의 유출물 위에 물을 분사하여 인화성 증기의 발화를 피할 것. 선창의 바닥으로 깨끗이 씻어 내릴 것. 다량의 물을 사용할 것. 펌프를 사용하여 선외로 배출시킬 것. 고체 : 유출물을 수거할 것. 선외로 배출시킬 것. 잔류물을 선창의 바닥으로 깨끗이 씻어 내릴 것. 다량의 물을 사용할 것. 펌프를 사용하여 선외로 배출시킬 것. 화물운송장치 (대량유출) : 선교와 거주구역이 바람이 불어오는 쪽으로 향하도록 할 것. 물분사장치를 사용하여 부식성이나 독성 증기를 몰아냄으로써 승무원과 거주구역을 보호할 것. 유출된 구역에 들어가는 말 것. 접근 금지. 무선으로 전문가의 조언을 구할 것. 전문가들에 의한 위험성 평가가 있은 후에 계속 진행할 것. 적절한 환기조치를 취할 것. 자장식 호흡구를 착용하지 아니하고는 유출된 구역에 들어가지 말 것. 들어가기 전에 대기상태를 점검할 것(독성 및 폭발 위험성). 만약 대기상태를 점검할 수 없을 시에는 들어가지 말 것. 증기가 증발되도록 둘 것. 접근 금지. 환기장치를 사용하는 경우에는 선박의 종사장소(예, 거주구역, 기관구역, 작업구역)로 독성증기나 퓸(fume)이 들어가는 것이 방지되도록 특별한 주의를 기울일 것. 액체 : 유출된 구역을 충분히 환기시킬 것. 유출물 표면에 물을 분사하여 인화성 증기의 발화를 방지할 것. 선창의 바닥으로 깨끗이 씻어 내릴 것. 다량의 물을 사용할 것. 펌프를 사용하여 선외로 배출시킬 것. 고체 : 유출물을 수거할 것. 선외로 배출시킬 것. 잔류물을 선창의 바닥으로 깨끗이 씻어 내릴 것. 다량의 물을 사용할 것. 펌프를 사용하여 선외로 배출시킬 것.
